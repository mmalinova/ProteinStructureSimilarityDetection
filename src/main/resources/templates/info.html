Welcome to "Protein Structure Similarity Detection"


Protein structure alignment algorithms are often time-consuming, resulting in challenges for large-scale protein structure similarity-based retrieval. There is an urgent need for more efficient structure comparison approaches as the number of protein structures increases rapidly.

The comparison algorithm used for Protein structure similarity detection here is End-Space Free Alignment with Geometric Properties Detection.
This is an alignment technique, which ignore the requirement that sequences start and end at the same place.
It allows aligning of sequences that overlap or include one another and is suitable for protein classification and similarity detection.
Time complexity of protein secondary structure similarity detection, using End-Space Free Alignment with Geometric Properties Detection is O(m,n).
