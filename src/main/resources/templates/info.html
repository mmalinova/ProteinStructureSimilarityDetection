Welcome to "Protein Structure Similarity Detection"

    Protein structure alignment algorithms are often time-consuming, resulting in challenges for large-scale protein structure similarity-based retrieval. There is an urgent need for more efficient structure comparison approaches as the number of protein structures increases rapidly.

    The comparison algorithm used for Protein structure similarity detection here is End-Space Free Alignment with Geometric Properties Detection.
This is an alignment technique, which ignore the requirement that sequences start and end at the same place.
It allows aligning of sequences that overlap or include one another and is suitable for protein classification and similarity detection.
Time complexity of protein secondary structure similarity detection, using End-Space Free Alignment with Geometric Properties Detection is O(m,n).

In order to use the algorithm you need to:

    1. Enter the Positive(the number that will be used if there is a match between the secondary structure elements) and Negative(the number that will be used if there is a difference between the secondary structure elements) scoring function type.
    2. Specify the two PDB URLs which describes the PDB files that will be compared. Please make sure that the URLs you provide are from the official PDB (Protein data bank) site and that all the information is at the PDB URLs. This will affect the final results!
      2.1 In order to get the right URL for your protein, you can type the name of the protein in the Search bar.
      2.2 After getting the right protein click on "Display Files" -> "PDB Format".
      2.3 Then copy the URL address and paste it to the field below the label "PDB File URL:".
    3. Click on the "Detect" button.

    After clicking on the "Detect" button the algorithm gets all the input information, processes it and calculate the Degree of similarity. Meanwhile, you can see 3D structure of the two uploaded protein PDB URLs. A library Jmol is used for this purpose.
